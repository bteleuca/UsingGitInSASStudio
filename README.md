

# Using Git in SAS Studio on SAS Viya

## Usage
Students should create a **Fork** of this repository.

## Versioning with SAS® Studio Git Integration

This repository contains the hands-on exercise materials for the **Versioning with SAS® Studio Git Integration** workshop. With Git becoming the industry standard for source-code management, this course teaches you how to effectively use Git with SAS Studio on the SAS Viya platform to streamline your software development projects.

**Course Code:** DE410W / DE410A
**Platform:** [learn.sas.com](https://learn.sas.com/course/view.php?id=7385)
**Estimated Duration:** 7.5 hours

## What You'll Learn

- Set up and manage Git profiles in SAS Studio
- Clone existing Git repositories and push changes to remote repositories
- Perform key Git operations: clone, pull, branch, stage, commit, merge, and push
- Work with feature branches: create, merge, and resolve merge conflicts
- Reset branches to previous commits and rebase for a cleaner project history
- Manage work in progress with Git stash functionality

## Target Audience

This workshop is designed for:
- SAS Studio users
- Data scientists
- Software developers
- IT professionals

Whether you are new to Git or have some familiarity with it, this workshop will help you manage and collaborate on your code more effectively within SAS Viya projects.

- Basic understanding of SAS Studio and the SAS Viya platform
- Familiarity with software development concepts
